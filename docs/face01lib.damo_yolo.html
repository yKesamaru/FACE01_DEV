face01lib.damo_yolo.base module

Summary:

このスクリプトは、DAMO-YOLOを使用して画像内の物体を検出するプログラムです。 指定された画像をモデルに入力し、検出された物体のクラス、スコア、バウンディングボックスを出力します。 スコアが0.5以上の検出結果のみを表示します。

class Infer(config, ckpt_path, infer_size=[640, 640], device='cuda', engine_type='torch')[ソース]

ベースクラス: object

画像内の物体を検出するための推論クラス。

config

モデルの設定ファイル。

ckpt_path

学習済みモデルのチェックポイントのパス。

infer_size

推論時の画像サイズ（デフォルトは640x640）。

device

推論を行うデバイス（デフォルトはCUDA）。

engine_type

推論に使用するエンジンタイプ（torch、onnx、tensorRTのいずれか）。

forward(image)[ソース]

画像を推論します。

パラメータ:

image -- 推論対象の画像（ファイルパスまたは numpy.ndarray）。

戻り値:

推論後のバウンディングボックス、スコア、クラスインデックスのリスト。

postprocess(preds, origin_shape)[ソース]

推論結果を後処理します。

パラメータ:

  • preds -- モデルの推論結果。

  • origin_shape -- 元画像の形状。

戻り値:

バウンディングボックス、スコア、クラスインデックスのリスト。

preprocess(image_path)[ソース]

画像を前処理します。

パラメータ:

image_path -- 処理対象の画像ファイルのパス。

戻り値:

前処理後の画像テンソルと元画像の形状。
