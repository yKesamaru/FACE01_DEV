version_control module

バージョンコントロールのためのスクリプト.

Summary:

バージョンを上げる場合、FACE01フォルダからDISTフォルダへ必要なファイルだけをコピーしたり Dockerイメージを作ってDockerHubへアップロードしたりSphinxでドキュメントを生成してDISTフォルダへ コピーしたり煩雑な作業があり、その手順をすっかり忘れてしまいました。 そこで上記工程をスクリプト化することにしました。

returns:

None
