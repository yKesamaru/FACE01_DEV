Installation

wget https://raw.githubusercontent.com/yKesamaru/FACE01_DEV/master/INSTALL_FACE01.sh
chmod +x INSTALL_FACE01.sh
bash -c ./INSTALL_FACE01.sh

NOTE: THIS IS ONLY USE FOR UBUNTU 20.04. If you are a Windows user, please use on WSLg or Docker. If you are using another Linux distribution, use Docker, Boxes, or lxd.

Alternatively, refer to INSTALL_FACE01.sh and install manually.
